\section{Características gerais}

O \textit{software} Simulador de difusão térmica 3D, é um software
programado sob o Paradigma Orientado ao Objeto em C++, capaz de simular
a transferência de calor em objetos tridimensionais, com formas e
superfícies complexas e definidas pelo usuário, o qual também pode
definir quais materiais constituem o objeto, e qual o método utilizado
para calcular as propriedades termofísicas para cada material. Também
é permitido a renderização 3D do objeto, e salvar os resultados em
formato pdf.

Para resolver o problema da transferência do calor, é modelado a equação
diferencial por diferenças finitas implícitas, especificamente pelo
método BTCS (\textit{Backward Time Centered Space}), que é incondicionalmente
estável. As condições iniciais, são inseridas pelo usuário, e as condições
de contorno externo são definitas por regiões que não trocam calor
com o meio externo (condição de contorno de Neumann).

\section{Especificação\index{especificação}}

Deseja-se desenvolver um software com interface gráfica amigável ao
usuário, onde seja possível desenhar o objeto 3D, por meio de perfis,
com o usuário escolhendo a temperatura e o material. A simulação é
governada pela Equação da Difusão Térmica, a qual é modelada por diferenças
finitas, pelo método BTCS, com fronteiras seladas. 

Na dinâmica de execução, o usuário deverá escolher o tamanho do objeto,
a temperatura, em qual perfil está desenhando, o material e suas propriedades
termofísicas, e um ponto de monitoramento e estudo. O usuário terá
a liberdade para utilizar um dentre três métodos para obter as propriedades
dos materiais: propriedades constantes, correlação e interpolação.

Após os desenhos do usuário e colocado o simulador para rodar, o simulador
irá calcular a temperatura iterativamente em cada ponto, e só passará
para o próximo tempo se o erro entre iterações for menor que um valor
aceitável. Posteriormente, o desenho será atualizado e mostrará a
nova distribuição de temperatura, e plotará os gráficos com os novos
valores calculados.

O software será programado em C++, com paradigma orientado ao objeto,
utilizando a biblioteca \textit{Qt} para criar a interface do usuário,
e \textit{qcustomplot }para gerar os gráficos.

Para calcular as propriedades termofísicas dos materiais, são utilizados
três modelos: propriedades constantes, por correlação e por interpolação.

Os principais termos e suas unidades utilizadas neste projeto estão
listadas abaixo:
\begin{itemize}
\item Dados relativos ao material:
\begin{itemize}
\item $c_{p}$ - capacidade térmica $\left[J/g\cdot K\right]$
\item $k$ - condutividade térmica $\left[W/m\cdot K\right]$
\item $\rho$ - massa específica $\left[kg/m^{3}\right]$
\end{itemize}
\item Dados relativos ao objeto
\begin{itemize}
\item $\Delta x,\Delta y$ - distância entre os centros dos blocos, valor
inicial: 1px=0.0026m $[m]$;
\item $\Delta z$ - distância entre perfis, valor inicial: 0.05m $[m]$;
\item $T$ - temperatura no nodo $[K]$;
\end{itemize}
\item Variáveis usadas na simulação:
\begin{itemize}
\item $i$ - posição do nodo em relação ao eixo x;
\item $k$ - posição do nodo em relação ao eixo y;
\item $g$ - qual grid/perfil está sendo analisado;
\item $t$ - tempo atual;
\item $\nu$ - número da iteração.
\end{itemize}
\end{itemize}
